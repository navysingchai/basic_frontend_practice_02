week 10 : ทำงานในคาบเป็น self practice อยู่ใน /practice01
    - ตอนแรกคือการเตรียมข้อมูลและโครงสร้างที่จำเป็นก่อนที่จะเริ่มเขียนพวกโค้ดต่าง ๆ จะมี
        defaultSetting: เป็น Object ที่เก็บ "ค่ามาตรฐาน" ของธีมเอาไว้ ถ้าผู้ใช้ยังไม่เคยตั้งค่าอะไรเลย ระบบจะใช้ค่าจากที่นี่
        settingKey: ใช้ Object.keys() เพื่อดึง key ทั้งหมด ("backgroundColor", "fontColor", "fontSize") มาเก็บในรูปแบบ Array เพื่อให้ง่ายต่อการวนลูป (loop) ในภายหลัง
        mappingStyle: เป็น Object ที่ทำหน้าที่เป็น "ตัวแปล" ระหว่าง ชื่อ id ของ input กับ ชื่อ property ของ CSS จริงๆ สังเกตว่า fontColor จะถูกแปลไปเป็น color ซึ่งเป็นชื่อ property ที่ถูกต้องใน CSS
    - มี setLocalStorage(key, value, notExist = false): เป็นfunction สำหรับบันทึกข้อมูลเข้า local storage
            ถ้า notExist เป็น true (ใช้ตอนเริ่มต้น) มันจะบันทึกค่าก็ต่อเมื่อ ยังไม่มีค่านั้นถูกบันทึกอยู่ก่อนแล้ว
            ถ้า notExist เป็น false (ใช้ตอนกด Save) มันจะ บันทึกทับค่าเก่า ทันที
    - ทำ removeLocalStorage(item): function สำหรับลบข้อมูลออกจาก LocalStorage (แม้ในโค้ดนี้จะยังไม่ได้เรียกใช้กับปุ่ม Reset ก็ตาม แต่ก็ทำไว้ก่อน เผื่อใช้งานอื่น อยู่ดี ๆ ก็คิดได้ เลยทำไว้)
    - setStyle(): function นี้ทำหน้าที่ อ่านค่า จาก LocalStorage แล้วนำมา กำหนดสไตล์ ให้กับ document.body โดยตรง มันจะวนลูปตาม mappingStyle เพื่ออ่านค่าและกำหนดสไตล์ให้ถูกคู่
    - เมื่อผู้ใช้เปิดหน้าเว็บนี้ครั้งแรก โค้ดส่วน init(); setStyle(); จะทำงานทันที
    - ส่วนสุดท้ายคือพวก event ต่าง ๆ
        ใช้ document.getElementById เพื่อเข้าถึงปุ่ม Save และ Reset
        btnSave.addEventListener('click', ...ใใใ): เป็นการ "ดักฟัง" เหตุการณ์เมื่อผู้ใช้คลิกปุ่ม Save และจะทำตามขั้นตอนต่อไปนี้:
            e.preventDefault(): ป้องกันไม่ให้ฟอร์มทำงานตามปกติ (ซึ่งก็คือการรีเฟรชหน้าเว็บ)
            วนลูป settingKey.forEach(...): วนลูปตาม key ทั้ง 3 ตัว ("backgroundColor", "fontColor", "fontSize")
            ดึงค่าใหม่: ภายในลูป จะใช้ document.querySelector เพื่อหา input ที่มี id ตรงกับ key นั้นๆ แล้วดึงค่า (.value) ที่ผู้ใช้เลือกออกมา
            บันทึกค่าใหม่: เรียก setLocalStorage(k, value, false) เพื่อบันทึกค่าใหม่ที่ผู้ใช้เลือกลงไป ทับของเก่า ใน LocalStorage
            แสดงผลทันที: หลังจากบันทึกค่าทั้งหมดเสร็จแล้ว จะเรียก setStyle() เพื่ออัปเดตสไตล์ของหน้าเว็บให้ตรงกับค่าใหม่ทันที โดยไม่ต้องรีเฟรช
